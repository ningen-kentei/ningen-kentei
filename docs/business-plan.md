## 1. エグゼクティブサマリー

**「AI時代、あなたはまだ"人間"ですか？」**

ニンゲン力検定は、AI時代における「人間力」と「社会不適合度（社不度）」を2軸で測定するWeb診断サービスである。全20問の設問に回答するだけで、レーダーチャート付きの診断結果と10種類の称号（「圧倒的人間」「完全社不」「異端の天才」など）が得られる。

自虐エンタメ路線を軸に、Z世代を中心としたSNSバイラルでの集客を狙う。「社不 x 人間力」という切り口は既存の診断サービスにない独自ポジションであり、競合不在のブルーオーシャンである。

**収益モデル**: 広告収入 + フリーミアム課金のハイブリッド型
**目標**: サービス開始6ヶ月で月間5万円の収益達成
**初期投資**: 約5,000円（ドメイン・サーバー代）

## 2. ミッション・ビジョン

## 3. 事業概要

### サービス内容
Web上で全20問の4択問題に回答し、「人間力スコア（0〜100点）」と「社不度（0〜100%）」の2軸で診断結果を表示するサービス。

### コンセプト
- **自虐エンタメ路線**: 笑いながら自分を知る。結果が「社不」でもポジティブ一言を必ず添える
- **2軸診断**: 人間力と社不度は独立して算出。「能力は高いのに社会に馴染めない」等の複雑な結果が出せる
- **AI時代の文脈**: 「AIに置き換えられない力」という時事性のあるテーマ

### サービスの特徴
1. **2軸の独立スコア**: 人間力が高くても社不度が高い、という「あるある」を表現できる
2. **レーダーチャート5項目**: 空気読み力・継続力・生活力・社会適応力・自走力の可視化
3. **10種類の称号**: 「量産型ニンゲン」から「異端の天才」まで、シェアしたくなるネーミング
4. **SNS最適化**: 結果画像の自動生成で、ワンタップでシェア可能

## 6. プロダクト詳細

### 診断の仕組み

**基本フロー**:
1. ユーザーがWebサイトにアクセス
2. 全20問の4択問題に回答（所要時間: 約3〜5分）
3. 回答をもとにスコアを算出
4. 結果画面を表示（称号 + レーダーチャート + ポジティブ一言）
5. SNSシェアボタンで拡散

**スコアリング**:
- **人間力スコア（0〜100点）**: レーダーチャート5項目（各0〜20点）の合計
- **社不度（0〜100%）**: 各問の社不ポイント（0〜5）の合計。人間力とは独立して算出
- 2軸が独立しているため、「能力は高いが社会に馴染めない」等のリアルな結果が実現

### 設問設計の概要

全20問を5カテゴリに分類:

| カテゴリ | 問番号 | 問数 | 主な影響項目 |
|---------|--------|------|------------|
| 生活習慣系 | Q1〜Q4 | 4問 | 生活力・継続力 |
| 空気読み系 | Q5〜Q8 | 4問 | 空気読み力・社会適応力 |
| 社会適合系 | Q9〜Q12 | 4問 | 社会適応力・主体性 |
| AI時代系 | Q13〜Q16 | 4問 | 主体性・空気読み力 |
| メンタル系 | Q17〜Q20 | 4問 | 継続力・主体性 |

**設問の特徴**:
- 全問4択でサクサク回答可能
- 選択肢がユーモラスで「あるある」感がある（例: 「ファブリーズが洗濯だと思っている」）
- 各選択肢は2〜3項目のレーダー項目に影響 + 独立した社不ポイント
- 全レーダー項目の最大到達可能点数が20点に調整済み

### レーダーチャート5項目

| 項目 | 定義 |
|------|------|
| 空気読み力（0〜20） | 場の雰囲気を察知し、適切に振る舞う力 |
| 継続力（0〜20） | 物事を習慣的に続け、コツコツやり抜く力 |
| 生活力（0〜20） | 規則正しい生活を自力で維持する力 |
| 社会適応力（0〜20） | 社会のルール・慣習・マナーに適応する力 |
| 主体性（0〜20） | 自分で考え、判断し、主体的に行動する力 |

### 結果タイプ10種

人間力スコア（3段階）x 社不度（3段階）+ 特殊パターン1つ = 全10タイプ

| タイプ | 称号 | 条件 |
|--------|------|------|
| TYPE-1 | 量産型ニンゲン | 人間力 0〜39 / 社不度 0〜39% |
| TYPE-2 | そこそこの一般人 | 人間力 40〜79 / 社不度 0〜39% |
| TYPE-3 | 圧倒的人間 〜AIが嫉妬するレベル〜 | 人間力 80〜100 / 社不度 0〜39% |
| TYPE-4 | ゆるふわ社不予備軍 | 人間力 0〜39 / 社不度 40〜69% |
| TYPE-5 | 器用な変人 | 人間力 40〜79 / 社不度 40〜69% |
| TYPE-6 | ハイスペ変人 | 人間力 80〜100 / 社不度 40〜69% |
| TYPE-7 | 完全社不 〜社会のバグ〜 | 人間力 0〜39 / 社不度 70〜100% |
| TYPE-8 | 社不界のエリート | 人間力 40〜79 / 社不度 70〜100% |
| TYPE-9 | 異端の天才 〜社会に収まりきらない逸材〜 | 人間力 80〜100 / 社不度 70〜100% |
| TYPE-10 | 完全なる凡人 〜パラメータ均等の奇跡〜 | 全5項目が8〜12点の範囲内（最優先判定） |

各タイプには自虐的だが愛のある説明文と、必ずポジティブな一言が添えられる。

### 傾向分析（実装済み）
結果ページではタイプ判定に加え、以下の5つの傾向をパーセンテージで表示する:
- **世渡り力**: 空気読み力と社会適応力から算出
- **SNS炎上リスク**: 空気読み力の低さと主体性から算出
- **留年しやすさ**: 継続力・主体性・生活力の低さから算出
- **一人暮らし耐性**: 生活力・主体性・継続力から算出
- **職場サバイバル力**: 継続力・社会適応力・主体性から算出

各項目にスコアに応じた3段階のコメントが付き、シェアしたくなる要素を強化している。

### 将来のアプリ展開（Phase 3）
- **空気読みダンジョンゲーム**: 診断結果の5パラメータを使ったRPG風ゲーム
- シチュエーション型の選択肢ゲームで、人間力パラメータがステータスとして反映
- リテンション向上とアプリ課金による収益拡大を狙う

## 7. ビジネスモデル

### 収益源

| 収益源 | 内容 | 導入時期 |
|--------|------|---------|
| 広告収入 | Google AdSense（結果画面中心に配置） | Phase 1（初月〜） |
| フリーミアム課金 | 詳細レポート（300〜500円） | Phase 2（3ヶ月目〜） |
| アプリ課金 | ゲーム内アイテム・追加診断 | Phase 3（6ヶ月目〜） |

### 価格設定

**無料コンテンツ（全ユーザー）**:
- 基本診断（20問）
- 称号 + レーダーチャート + ポジティブ一言
- SNSシェア用画像

**有料コンテンツ（300〜500円の買い切り）**:
- 各レーダー項目の詳細分析
- タイプ別の「人間力向上アドバイス」
- AI時代の適職診断
- 友達との相性診断

### 収益シミュレーション

**前提条件**:
- 広告RPM: 300円（診断系サイトの平均的な値）
- 課金CVR: 1%（無料ユーザーのうち課金する割合）
- 課金単価: 300円

**Phase 1（1〜3ヶ月目）: 広告のみ**

| 月 | 月間PV | 広告収入 | 合計 |
|----|--------|---------|------|
| 1ヶ月目 | 5,000 | 1,500円 | 1,500円 |
| 2ヶ月目 | 15,000 | 4,500円 | 4,500円 |
| 3ヶ月目 | 30,000 | 9,000円 | 9,000円 |

**Phase 2（4〜6ヶ月目）: 広告 + 課金**

| 月 | 月間PV | 診断完了ユーザー | 広告収入 | 課金収入 | 合計 |
|----|--------|----------------|---------|---------|------|
| 4ヶ月目 | 50,000 | 12,500人 | 15,000円 | 37,500円 | 52,500円 |
| 5ヶ月目 | 70,000 | 17,500人 | 21,000円 | 52,500円 | 73,500円 |
| 6ヶ月目 | 100,000 | 25,000人 | 30,000円 | 75,000円 | 105,000円 |

※診断完了ユーザー = PVの約25%と想定（1人あたり約4PV: トップ→診断→結果→シェア）
※課金収入 = 診断完了ユーザー x CVR 1% x 300円

**現実的な目標（ハイブリッドモデル）**:
- 月5万円達成の目安: 月間約5万PV（広告1.5万 + 課金3.5万）
- これは約12,500人の診断完了ユーザーに相当

## 8. マーケティング戦略

### チャネル別施策

#### X（Twitter）- メインチャネル
- **ハッシュタグ戦略**: #ニンゲン力検定 でトレンド入りを狙う
- **投稿テンプレ**: 結果画像に「あなたも診断してみる？」のCTAを自動挿入
- **ネタ投稿**: 各タイプの面白い解説を定期投稿（週3回）
- **目標**: 初月1,000シェア、3ヶ月で10,000シェア

#### TikTok - バイラル狙い
- **「やってみた」動画**: インフルエンサーに診断してもらう動画を促進
- **リアクション動画**: 結果を見た時のリアクションが面白いコンテンツに
- **目標**: 3ヶ月以内に1本10万再生以上の動画を出す

#### Instagram - ビジュアル訴求
- **ストーリーズ最適化**: 結果画像をストーリーズサイズ（9:16）で自動生成
- **シェアしやすいデザイン**: パステルカラー + 大きな称号テキスト
- **目標**: 月間1,000ストーリーズシェア

#### SEO - 長期集客
- **狙うキーワード**:
  - 「人間力 診断」（月間検索数: 約500）
  - 「社会不適合者 診断」（月間検索数: 約1,200）
  - 「AI 仕事 診断」（月間検索数: 約300）
- **コンテンツSEO**: 各タイプの解説記事を個別ページで公開
- **目標**: 6ヶ月以内に主要キーワードで検索10位以内

### バイラル設計
1. **結果画像の自動生成**: 称号 + レーダーチャート + 社不度を1枚の画像に
2. **シェアボタンのワンタップ化**: X・Instagram・LINEの3プラットフォーム対応
3. **比較機能**: 「友達の結果と比較する」で再訪問を促進
4. **週替わりランキング**: 最も多いタイプをリアルタイム表示

| タスク | 自作/外注 | 理由 | 想定コスト |

## 13. リスク分析と対策

### リスクマトリクス

```
影響度 大 |  [R3]        |  [R1]        |
          |              |              |
影響度 中 |  [R5]        |  [R2]        |
          |              |              |
影響度 小 |  [R4]        |  [R6]        |
          +--------------+--------------+
          発生確率 低      発生確率 高
```

### リスク詳細と対策

#### R1: 一過性バズで終わる（発生確率: 高 / 影響度: 大）
- **内容**: バズった後にPVが急落し、収益が維持できない
- **対策**:
  - 月1回の新診断テーマ追加（季節・時事ネタ）
  - SEOでの安定的な流入基盤を構築
  - メールアドレス or LINE登録で再訪問を促す
  - ゲーム要素追加でリテンション向上

#### R2: PVが思うように伸びない（発生確率: 高 / 影響度: 中）
- **内容**: そもそもバズらず、月間1万PV以下が続く
- **対策**:
  - 小額（5,000円程度）のマイクロインフルエンサー施策
  - 設問内容の改善（ユーザーフィードバック反映）
  - Xでの地道な投稿継続（結果タイプの解説など）
  - 他の診断系メディアへの相互リンク

#### R3: 「不安を煽る」で炎上する（発生確率: 低 / 影響度: 大）
- **内容**: 「社会不適合者」というワードが差別的と批判される
- **対策**:
  - 自虐エンタメトーンを徹底（上から目線にならない）
  - 全結果に必ずポジティブな一言を添える
  - 「社不」はネガティブではなく個性として表現
  - 炎上時の対応テンプレートを事前に準備

#### R4: 技術的な問題（発生確率: 低 / 影響度: 小）
- **内容**: サイトダウン、表示崩れ、スコア計算のバグ
- **対策**:
  - Vercelの自動スケーリングに任せる
  - リリース前に全パターンのテスト実施
  - ユーザーからのバグ報告フォームを設置

#### R5: 課金CVRが想定以下（発生確率: 低 / 影響度: 中）
- **内容**: 有料レポートに価値を感じてもらえず、CVR 0.5%以下
- **対策**:
  - 無料版でも十分楽しめるが、有料版で「もっと知りたい」を刺激する設計
  - 価格のA/Bテスト（300円 vs 500円）
  - 広告収入をメインにし、課金はボーナスと位置づける

#### R6: 類似サービスの出現（発生確率: 高 / 影響度: 小）
- **内容**: 似たコンセプトの診断サービスが登場する
- **対策**:
  - 先行者利益を活かしてブランドを確立
  - 設問の質とユーモアで差別化
  - コミュニティ形成で囲い込み
