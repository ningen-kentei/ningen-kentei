# ニンゲン力検定 - 診断ロジック・スコアリング設計書

---

## 1. 2軸スコア設計

### 人間力スコア（0〜100点）
- レーダーチャート5項目の合計点
- 各項目0〜20点 x 5項目 = 最大100点

### 社不度（0〜100%）
人間力スコアとは**独立して算出**する。単純な逆相関ではない。

#### 社不度の算出方法

各設問の選択肢に「社不ポイント」（0〜5）を個別に設定する。

- **社不ポイントの合計**（全20問、最大100ポイント）がそのまま社不度（%）となる
- 社不ポイントは人間力の配点とは独立して設計する

#### 2軸が独立する仕組み

| パターン | 例 |
|---|---|
| 人間力高 × 社不度低 | 空気も読めて生活力もある、社会適合型の人間 |
| 人間力高 × 社不度高 | 能力は高いが社会のルールに従わない、異端型 |
| 人間力低 × 社不度低 | 能力は低いが従順、おとなしく社会に溶け込む型 |
| 人間力低 × 社不度高 | 能力も低く社会にも馴染めない、完全社不型 |

**設計のポイント**: 「自走力」や「空気読み力」が高くても、その使い方が反社会的・独自路線なら社不度は高くなる。逆に能力が低くても、素直に社会に従っていれば社不度は低い。

## 2. レーダーチャート5項目の定義

| 項目 | 定義 | 高い人の特徴 | 低い人の特徴 |
|---|---|---|---|
| **空気読み力**（0〜20） | 対人関係で場の雰囲気を察知し、適切に振る舞う力 | 相手の気持ちがわかる、場を壊さない | KY、自分の世界に没頭しがち |
| **継続力**（0〜20） | 物事を習慣的に続け、コツコツやり抜く力 | 三日坊主にならない、ルーティンが得意 | 飽き性、すぐ新しいものに飛びつく |
| **生活力**（0〜20） | 規則正しい生活を自力で維持する力 | 早寝早起き、自炊、整理整頓 | 昼夜逆転、部屋が散らかりがち |
| **社会適応力**（0〜20） | 社会のルール・慣習・マナーに適応する力 | 報連相ができる、締切を守る | ルールに反発、集団行動が苦手 |
| **主体性**（0〜20） | 自分で考え、判断し、主体的に行動する力 | 指示待ちにならない、自分の意見がある | 受動的、他人任せ |

## 4. 配点の仕組み

### 基本ルール
- 各問4択（A/B/C/D）
- 各選択肢は**2〜3項目**のレーダー項目に影響する
- 各選択肢の人間力配点は合計4〜8ポイント（項目ごとに0〜4を振り分け）
- 各選択肢に独立した社不ポイント（0〜5）を設定

### 配点設計の目安

各レーダー項目の**最大到達可能点数**が20になるよう調整する。

| レーダー項目 | 主に影響するカテゴリ | 副次的に影響するカテゴリ |
|---|---|---|
| 空気読み力 | 空気読み系（Q5〜Q8） | AI時代系（Q13〜Q16） |
| 継続力 | 生活習慣系（Q1〜Q4） | メンタル系（Q17〜Q20） |
| 生活力 | 生活習慣系（Q1〜Q4） | 社会適合系（Q9〜Q12） |
| 社会適応力 | 空気読み系（Q5〜Q8）、社会適合系（Q9〜Q12） | - |
| 主体性 | 社会適合系（Q9〜Q12）、AI時代系（Q13〜Q16） | メンタル系（Q17〜Q20） |

### 配点テンプレート例（1問分）

**Q5「飲み会で知らない人ばかりのテーブルに座った。あなたは？」**（空気読み系）

| 選択肢 | 空気読み力 | 社会適応力 | 社不ポイント |
|---|---|---|---|
| A: 自分から話しかけて場を盛り上げる | +4 | +3 | 0 |
| B: ニコニコしながら相槌を打つ | +3 | +2 | 1 |
| C: スマホをいじって時間をつぶす | +0 | +0 | 4 |
| D: 即座にトイレに避難して帰りのタイミングを計る | +1 | +0 | 5 |

※ 選択肢Dは「状況判断力」として空気読み力+1（逃げるという判断ができる）だが社不度は高い

### 各項目の得点レンジ設計

各レーダー項目が0〜20の範囲に収まるよう、以下の配分で設計する：

| レーダー項目 | 主要カテゴリからの最大得点 | 副次カテゴリからの最大得点 | 合計最大 |
|---|---|---|---|
| 空気読み力 | 12（Q5〜Q8から各最大3） | 8（Q13〜Q16から各最大2） | 20 |
| 継続力 | 12（Q1〜Q4から各最大3） | 8（Q17〜Q20から各最大2） | 20 |
| 生活力 | 14（Q1〜Q4から各最大3.5） | 6（Q9〜Q12から各最大1.5） | 20 |
| 社会適応力 | 10（Q5〜Q8から各最大2.5） | 10（Q9〜Q12から各最大2.5） | 20 |
| 主体性 | 8（Q9〜Q12から各最大2） | 8（Q13〜Q16）+ 4（Q17〜Q20） | 20 |

※ 実装時は整数に丸めて調整する

### 社不ポイントの設計方針

- 各問の社不ポイント最大は5
- 20問 x 最大5 = 最大100 → そのまま社不度%になる
- 社不ポイントが高い選択肢の例：
  - 「人に会いたくない」「ルールを無視する」「昼夜逆転」→ 高い（4〜5）
  - 「適度に社交的」「ルールは守る」→ 低い（0〜1）
  - 「能力は高いが我が道を行く」→ 中程度（2〜3）

## 5. 結果タイプ（10パターン）

### 判定マトリクス

人間力スコアを3段階、社不度を3段階+特殊パターン1つの計10タイプ。

```
社不度(%)  0    20    40    60    80    100
          |-----|-----|-----|-----|-----|
人間力     低(0-39)      中(40-69)      高(70-100)
80-100  |  TYPE-3   |  TYPE-6    |  TYPE-9    |
        |           |            |            |
40-79   |  TYPE-2   |  TYPE-5    |  TYPE-8    |
        |           |            |            |
0-39    |  TYPE-1   |  TYPE-4    |  TYPE-7    |
          社不度低      社不度中       社不度高
          (0-39%)      (40-69%)      (70-100%)
```

+ 特殊パターン TYPE-10（全項目がほぼ均等 = 各項目8〜12点の範囲内）

### 10タイプ詳細

#### TYPE-1: 「量産型ニンゲン」
- **条件**: 人間力 0〜39 / 社不度 0〜39%
- **説明**: 能力は控えめだけど、社会には何とか溶け込んでいる。目立たないが害もない、まさに量産型。AIに仕事を奪われる最有力候補かもしれないが、少なくとも職場の飲み会には呼ばれるタイプ。
- **ポジティブ一言**: 「協調性は立派な才能です。あなたがいるからチームが回っている。」

#### TYPE-2: 「そこそこの一般人」
- **条件**: 人間力 40〜79 / 社不度 0〜39%
- **説明**: バランスよく人間をやれている、THE普通の人。特筆すべき点がないのが最大の特徴。でも考えてみてほしい、「普通」でいられることがどれだけ難しい時代か。
- **ポジティブ一言**: 「普通であることは最強の生存戦略。あなたはしっかり人間をやれている。」

#### TYPE-3: 「圧倒的人間 〜AIが嫉妬するレベル〜」
- **条件**: 人間力 80〜100 / 社不度 0〜39%
- **説明**: 人間力MAX、社会適合度も高い、完璧超人。もはや人間のお手本。AIがどれだけ進化しても、あなたの代わりにはなれない。むしろAIの方が「こういう人間になりたかった」と思ってるかも。
- **ポジティブ一言**: 「文句なしのトップスコア。人間やめないでください、世界が困ります。」

#### TYPE-4: 「ゆるふわ社不予備軍」
- **条件**: 人間力 0〜39 / 社不度 40〜69%
- **説明**: 能力は低めで社会からもじわじわ離脱中。まだ完全な社不ではないが、このままいくと危ない。とりあえず朝起きることから始めよう。
- **ポジティブ一言**: 「自分のペースを大事にできる人は、実はレアな才能の持ち主。」

#### TYPE-5: 「器用な変人」
- **条件**: 人間力 40〜79 / 社不度 40〜69%
- **説明**: そこそこの能力があるのに、なぜか社会からちょっとズレている。器用なのに不器用、矛盾の塊。飲み会では面白い人枠に入るが、二次会には誘われないタイプ。
- **ポジティブ一言**: 「ズレているからこそ見える景色がある。その個性は武器になる。」

#### TYPE-6: 「ハイスペ変人」
- **条件**: 人間力 80〜100 / 社不度 40〜69%
- **説明**: 能力はめちゃくちゃ高いのに、どこか社会と噛み合わない。高性能なのにOSが独自規格、みたいな人。仕事はできるけど上司に嫌われるタイプ。
- **ポジティブ一言**: 「あなたが合わないのは社会の方。その実力があれば道は自分で作れる。」

#### TYPE-7: 「完全社不 〜社会のバグ〜」
- **条件**: 人間力 0〜39 / 社不度 70〜100%
- **説明**: 人間力低め、社不度MAX。社会というゲームのルールブックを読まずにプレイしている状態。でも大丈夫、インターネットがある時代に生まれてよかったね。
- **ポジティブ一言**: 「社会不適合は、社会の側が狭いだけかもしれない。自分の世界を大切に。」

#### TYPE-8: 「社不界のエリート」
- **条件**: 人間力 40〜79 / 社不度 70〜100%
- **説明**: それなりに能力はあるのに、社会に全然適合できていない。やればできる子なのに、やらない。社不としてはプロフェッショナルレベル。リモートワークの発明に最も感謝している人種。
- **ポジティブ一言**: 「あなたに合った環境が必ずある。今の時代、適合しなくても生きていける。」

#### TYPE-9: 「異端の天才 〜社会に収まりきらない逸材〜」
- **条件**: 人間力 80〜100 / 社不度 70〜100%
- **説明**: 圧倒的な人間力を持ちながら、社会のルールに収まらない。歴史上の偉人にこのタイプが多い（と自分に言い聞かせよう）。天才と変人は紙一重、あなたは両方。
- **ポジティブ一言**: 「枠にはまらない人間こそが世界を変える。あなたはその可能性を持っている。」

#### TYPE-10: 「完全なる凡人 〜パラメータ均等の奇跡〜」
- **条件**: レーダーチャート全5項目が8〜12点の範囲内（かつ他のタイプ判定より優先）
- **説明**: 全パラメータが驚くほど均等。尖りがなさすぎて逆に尖っている。RPGなら「勇者」の初期ステータス。何にでもなれるが、何者にもなれていない。ある意味で最もレアなタイプ。
- **ポジティブ一言**: 「バランス型は最も伸びしろがある。どの方向にも進化できるのは最大の強み。」

### タイプ判定の優先順位
1. まずTYPE-10の条件（全項目8〜12点）をチェック → 該当すればTYPE-10
2. 該当しなければ、人間力スコアと社不度の組み合わせでTYPE-1〜9を判定

## 6. スコア計算フロー（まとめ）

```
ユーザーが全20問に回答
    |
    v
各回答の選択肢から配点を取得
    |
    +---> レーダー5項目の得点を合算 → 各項目0〜20点
    |         |
    |         v
    |     人間力スコア = 5項目の合計（0〜100）
    |
    +---> 社不ポイントを合算 → 社不度（0〜100%）
    |
    v
TYPE-10条件チェック（全項目8〜12点？）
    |
    +-- YES → TYPE-10
    |
    +-- NO → 人間力スコア × 社不度のマトリクスでTYPE-1〜9判定
    |
    v
結果表示: 称号 + 説明 + レーダーチャート + 傾向分析 + ポジティブ一言
```

## 7. 傾向分析（実装済み）

結果ページにて、レーダーチャートの5項目スコアを組み合わせて以下の傾向をパーセンテージで表示する。各項目は3段階のコメント付き。

| 傾向項目 | 算出ロジック | 備考 |
|---------|------------|------|
| 世渡り力 | 空気読み×0.4 + 社会適応力×0.4 + 継続力×0.2 | ポジティブ代表 |
| SNS炎上リスク | (100-空気読み)×0.5 + (100-社会適応力)×0.3 + 主体性×0.2 | 主体性高いと「言いたいこと言う」 |
| 留年しやすさ | (100-継続力)×0.4 + (100-主体性)×0.3 + (100-生活力)×0.3 | 自虐枠 |
| 一人暮らし耐性 | 生活力×0.5 + 主体性×0.3 + 継続力×0.2 | 実用感あり |
| 職場サバイバル力 | 継続力×0.4 + 社会適応力×0.3 + 主体性×0.2 + 空気読み×0.1 | 仕事系 |

※各スコアは0〜20を0〜100に正規化して計算、結果は0〜100%にクランプ

## 8. 実装時の注意事項

1. **配点バランスの検証**: 全選択肢をランダムに選んだ場合、各レーダー項目が10点前後（中央値）になるよう調整する
2. **社不度と人間力の相関**: テストデータで相関係数を確認し、|r| < 0.5程度になるよう社不ポイントを調整する
3. **TYPE-10の出現率**: 全項目8〜12はかなり厳しい条件なので、レア枠として適切。出現率が低すぎる場合は範囲を7〜13に広げることも検討
4. **端数処理**: 全て整数で処理し、小数点は使わない
